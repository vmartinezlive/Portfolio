Project

The list below are links to all the projects from the first week of the program in Epicodus. These projects are colloborated with different students whom I paired with during the week.

  • My first Webpage

    • First project of the week is where we learned how to create files. Write on html and learn each tags to use.

  • Branching

    • This project explores the steps on how to brach from master copy. Create new branch, how to work and navigate between branches. We learned the benefits of branching by experimenting with new codes without affecting or making changes to the master file until project is finalized.

  • Animal-Shelter

    • In this project we explored the use of commits and push to remote repository on GitHub. It also showcase the practice of changing the styles of elements on the page according to cascading precedence.

  • Wiki

    • This projects will showcase how we were able to rebuild Wikipedia homepage using using divs, spans, classes, floats, to style the page.
